AssetIQ Data Grids

AssetIQ is a system, developed by Crayon, for loading asset data into Flexera's FNMS Asset Management system.
For more information about AssetIQ, please refer to the help pages linked of the Index page

AssetIQ Data Grid

AssetIQ pages use data grids to allow user to record and track the progressing of uploaded data. Data grids are alo used to allow uesers to inspect staged data.

In the example below, a grid on the Criticality Uploads page shows AssetIQ with two uploaded Asset Criticality data files.
The first file (FID = 1000) has been staged and been processed for validation, but the validation checks have reported 12 validation errors.


Filter Rows

All AssetIQ data grids show a Filter Row button. The Filter Row button toggles the grid to display a filter row that allows users to apply a text match filter on each grid column.

The filter row supports the the following text match modifiers:

  • Begins with
  • Contains
  • Doesn't contain
  • Ends with
  • Equals
  • Doesn't equal

Column Chooser

All AssetIQ data grids show a Column Chooser button. The Column Chooser button toggles the grid to display a Column Chooser dialog.
Additional columns can be dragged to and from the Columns Chooser dialog and the grid.


Selected Rows

AssetIQ grids have been configured to allow single row only selection. The selected row is shown with a orange hilight.
All grid data processing actions are performed on the selected row.
