AssetIQ Help Index Page

AssetIQ is a system, developed by Crayon, for loading asset data into Flexera's FNMS Asset Management system.
For an introduction to AssetIQ, please refer to the Introduction page

Overview

The following help pages provide an overview of the AssetIQ system:

  • Glossary: A glossary of terms used in AssetIQ
  • Introduction page: An introduction to AssetIQ
  • AssetIQ Page Map: Provides a map and guide to the User-level pages of AssetIQ
  • AssetIQ Manager and Admin Page Map: Provides a map and guide to the Manager and Admin-level pages of AssetIQ
  • User Interface Elements: This page describes some of the common user interface elements of AssetIQ
  • Data Grids: This page describes some of the common features and functions associated with data grids in AssetIQ
