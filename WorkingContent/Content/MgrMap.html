The AssetIQ Page Map

AssetIQ is a system, developed by Crayon, for loading asset data into Flexera's FNMS Asset Management system.
For more information about AssetIQ, please refer to the help pages linked of the Index page

Page Map (Admin and Manager pages)

Managers and Administrators can navigate to any of Manager level pages shown in the following page map by clicking on the corresponding entry in the navigation bar (see below).
Administrators can additionally navigate to the Admin level pages.

The follow top-level (landing) pages are used:

  • Default.aspx
    The default/login page that is show when a new AssetIQ sesson is started
  • Home.aspx
    A page that can be return to from any other page in the system. The Home page displays some simple metrics about the number of uploads performed over for 7, 90, and 365-day periods
  • Manager.aspx
    The Manager level landing page. The Manager page also displays some simple metrics about the number of uploads performed over for 7, 90, and 365-day periods
  • Admin.aspx
    The Amin level landing page. The Admin page includes some additional main menu options, and displays the same simple metrics ast the Home and Manager pages
  • Timeout.aspx
    A page that is displayed after a timout on any other page. The timeout page is similar to the Default page but automatically enforces user re-authentication

The following Manager level pages are are available to Asset-Managers, and Administrators:

  • TemplateUploads.aspx
    Manager/TemplateUploads.aspx: The TemplateUploads page allows Managers and Administrators to upload updates to spreadsheet templates
  • Users.aspx
    Manager/Users.aspx: The Users page allows Managers and Administrators to to view and edit User (registration) records
  • Lists.aspx
    Manager/Lists.aspx: The Lists page allows Managers and Administrators to to view and edit (Validation) List records

The following Admin level pages are are available to Administrators:

  • UserUploads.aspx
    Admin/UserUploads.aspx: The UserUploads page allows Administrators to upload spreadsheets containing user registration records
  • FileManager.aspx
    Admin/FileManager.aspx: The File Manager page gives Administrators file level access to all uploaded files
  • ProcessLogs.aspx
    Admin/ProcessLogs.aspx: The ProcessLogs page allows Administrators to view all logs records
