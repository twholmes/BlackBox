The AssetIQ Page Map

AssetIQ is a system, developed by Crayon, for loading asset data into Flexera's FNMS Asset Management system.
For more information about AssetIQ, please refer to the help pages linked of the Index page

Page Map (User and Site Auditor pages)

Users can navigate to any of User and Site Auditor level pages shown in the following page map by clicking on the corresponding entry in the navigation bar (see below).

The follow top-level pages are available to all users:
  • Default.aspx
    The default/login page that is show when a new AssetIQ sesson is started
  • Home.aspx
    A "landing" page that can be return to from any other page in the system. The Home page displays some simple metrics about the number of uploads performed over for 7, 90, and 365-day periods
  • Timeout.aspx
    A page that is displayed after a timout on any other page. The timeout page is similar to the Default page but automatically enforces user re-authentication

The follow "upload" pages are are available to Site-Auditors, Asset-Managers, and Administrators:
  • CriticalityUploads.aspx
    RioTinto/CriticalityUploads.aspx: The uploads page for Asset Criticality spreadsheets
  • SIDAUploads.aspx
    RioTinto/SIDAUploads.aspx: The uploads page for SIDA spreadsheets

The follow "open and inspect" pages are are available to Site-Auditors, Asset-Managers, and Administrators:
  • Spreadsheet.aspx
    The Spreadsheet (open) page opens an uploaded spreadsheet for viewing and editing of its content
  • InspectSiteAudit.aspx
    RioTinto/InspectSiteAudit.aspx: The dataset record inspect page for the SiteAudit dataset
  • InspectSIA.aspx
    RioTinto/InspectSIA.aspx: The dataset record inspect page for the SIA dataset
  • InspectSIDA.aspx
    RioTinto/InspectSIDA.aspx: The dataset record inspect page for the SIDA dataset
