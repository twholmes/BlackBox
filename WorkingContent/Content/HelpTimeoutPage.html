AssetIQ Timeout Page

AssetIQ is a system, developed by Crayon, for loading asset data into Flexera's FNMS Asset Management system.
For more information about AssetIQ, please refer to the help pages linked of the Index page

Page Description

The AssetIQ Timeout page is displayed whenever a session timeout occurs. Session timeouts occur when a page has been displayed without user activity for more than 14400 seconds.


The Timeout Page displays a form with the following controls:

  • User full-name
    		•  The user's full name is displayed above the User image
  • User image
    		•  A small photo/image or avatar identifying the user
  • SAMAccountName
    		•  The user account name (SAMAccountName)
  • Domain name
    		•  The (flat) domain name associated with the user account
  • User ID
    		•  AssetIQ's user-ID for the user
  • Home (button)
    		•  Clicking on the Home button returns the user to the Home page.

    Right-side Popout Panel

    The Timeout page's right-side popout panel contains two tabbed panes:

    My-Profile Tab

    The My Profile pane shows additional informations about the authenticated user. All users and update their user details, and Manager's can update user roles.


    Users Tab

    The Users pane contains a grid listing all users. Managers can select a user record and click Impersonate to impersonate that user while using the system.
