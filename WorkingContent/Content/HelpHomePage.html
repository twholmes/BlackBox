AssetIQ Home Page

AssetIQ is a system, developed by Crayon, for loading asset data into Flexera's FNMS Asset Management system.
For more information about AssetIQ, please refer to the help pages linked of the Index page

Page Description

The AssetIQ Home page is is a page that can be returned to from all other pages in AssetIQ, by clicking on the main menu Home button.


The Home Page displays an Uploads Dashboard with the following controls:

  • Chart: A bar chart showing uploaded file counts over 7, 90, and 365 day periods
  • Grid: The grid with records lising uploaded file counts for support datasets over 7, 90, and 365 day periods

Right-side Popout Panel

The Home page's right-side popout panel contains one tabbed pane:

File Manager Tab

The File Manager pane shows a File Manager control that allows Managers direct access to AssetIQ's file system. Managers can upload and download all AssetIQ managed datafiles.
