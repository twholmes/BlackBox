AssetIQ User-Interface Elements

AssetIQ is a system, developed by Crayon, for loading asset data into Flexera's FNMS Asset Management system.
For more information about AssetIQ, please refer to the help pages linked of the Index page

The AssetIQ Navigation Bar

The AssetIQ (page) navigation bar is found on the right panel of the AssetIQ portal. A Home page link button is always displayed for all users as the first menu item in the navbar. Other menu items are displayed dependant on the users role.

  • The page nav-bar can be expanded or collapsed by clicking on the panel-toggle button     to its right

The Breadcrumbs Menu

The Breadcrumbs Menu is a UI navigation aid to help the user traverse the AssetIQ page map

The Breadcrumbs Menu has two elements, seperated by " > ":

  • Page Group: A hyperlink to the home page for the group of pages to which the the displayed page belongs
  • Page Name: The name of the displayed page

Tabbed Pages

AssetIQ pages are implemenetd with one or more tabbed panels holding the main body of each page's content.


The example shown here is of the Criticality Uploads page with two tabbed panels:

  • Uploaded Files: The main panel for this page containing a data grid, data source action buttons, and a file upload control
  • Processing & Validation Results: A supplementary panel that allows the user to review data source validation results

User Menu

The User Menu is a drop-down menu that is shown on the top righthand side of each page. The User Menu provides simple visual confirmation of the user's identity.
The User Menu also support an option to allow the user to reauthenticate with the AssetIQ portal system.


The drop-down menu items are:

  • User name and icon: The authenticated user's name and an icon that can either be their selected avatar, or a thumbnail image of the user
  • Reauthenticate user: A button that can be used to force reauthentication of the user

Right-hand-side Popout Panel

The User Menu is a drop-down menu that is shown on the top righthand side of each page. The User Menu provides simple visual confirmation of the user's identity.
The User Menu also support an option to allow the user to reauthenticate with the AssetIQ portal system.


In this example, the right-hand-side popout panel for Criticality Uploads page is show expanded with two tabbed panels:

  • History: Contains a grid that shows a records of actions with results taken on the page
  • Validation Lists: Contains a grid that shows all Validation list records that apply to Criticality data sources
