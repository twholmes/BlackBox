AssetIQ Glossary

AssetIQ is a system, developed by Crayon, for loading asset data into Flexera's FNMS Asset Management system.
For more information about AssetIQ, please refer to the help pages linked of the Index page

Alphbetical list of terms

The following terms are referenced in AssetIQ help documentation:

  • Archived
    Status of a data source after its has been archived. Archived files are stored in AssetIQ for a further 90-days after archiving but are no longer available for processing
  • Dataset
    A data table holding asset data records of the same record type and schema
  • Dataset update
    An update to a dataset. Datasets a revision tracked but typically such tracking is not visible to end users
  • DataSource
    A source of asset data. In this release of AssetIQ, all data sources are spreadsheets
  • DataSource instance
    An instance of a data source. May be a copy or modified revision of a previously uploaded spreadsheet
  • Processed
    Status of a data source after it has been uploaded to FNMS in an FNMS Adapter run
  • Staged
    Status of a data source after it's data has been loaded into AssetIQ's "staging tables"
  • Template
    AssetIQ manages a set of spreadsheet templates, one for each registered data source. These template files can be downloaded by clicking on a hyperlink on an Uploads page
  • Uploaded
    Status of a data source after its spreadsheet has been uploaded to AssetIQ. AssetIQ retains all uploaded spreadsheets in its uploads file-store untill they are archived
  • Validated
    Status of a data source after it has passed all validation tests. ie No errors, but warnings are permitted
